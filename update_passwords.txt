MySQL ROOT:
  sudo mysql -u root -p
  <old password - default MySQL_root_password>
  
  SET PASSWORD=PASSWORD('<your new root password>');
  
  exit;
  
MySQL Observium:
  sudo mysql -u observium -p
  <old password - default MySQL_observium_password>
  
  SET PASSWORD=PASSWORD('<your new observium password>');
  
  exit;
  
  sudo vim /opt/observium/config.php
  
  change the line $config['db_pass'] with the same password as above.
  
Observium account login
  Open your browser and navigate to the Observium address, login.
  
  Go to the cogwheel menu, go to 'My Settings'.
  
  Change your password.
